👑 RCCG Kings World, Ile-Ife — Landing Page Spec (current build)

Vibe & Aesthetic:
- Pure black background (#000000) everywhere, no gradients on the base surface.
- Burgundy is the only accent color. No gold anywhere.
  --burgundy: #7b1e3a
  --burgundy-2: #a81e4f
- Text is white with dimmed variants (rgba white). Hairlines are 1px rgba(255,255,255,0.10).
- Typography: Inter for UI text (via next/font/google), Fraunces variable serif for display headings. Normal sizes (text-3xl / sm:text-4xl for section headings, text-4xl / sm:text-5xl for the hero h1). No massive display type.
- Real church website inspiration: VOUS, Elevation, Passion City. Minimal, no marquees, no scroll-reveal animations, no over-designed effects.

Tech stack:
- Next.js 16 App Router with Turbopack.
- React 19.
- Tailwind CSS v4 (via @tailwindcss/postcss).
- three.js + @react-three/fiber + @react-three/drei for the hero centerpiece.

Folder layout (route groups so future sections can slot in):
  app/
    layout.tsx                       (shared root layout, Inter + Fraunces fonts)
    globals.css                      (shared theme + keyframes)
    (landing)/
      page.tsx                       (URL "/")
      _components/
        Nav.tsx, Hero.tsx, HeroThree.tsx, ThreeScene.tsx,
        About.tsx, Community.tsx, Sermons.tsx, Give.tsx,
        MapSection.tsx, Footer.tsx,

Copy & style rules:
- No middle-dots (·) or em-dashes (—) as separators in body copy. Plain sentences.
- No AI-flavored phrasing ("vibe", "cinematic", "breathtaking", "radical"). Church-native, warm, direct.
- Only pure black backgrounds. Only burgundy as accent. No gold, no blue.
- No entrance / scroll-reveal animations. The only motion allowed is:
  1. Three.js icosahedron rotation in the hero.
  2. Scroll cue slide-down in the hero.
  3. Nav background fade on scroll.
  4. Hover states (button colors, card border, image scale, link underline).
